# TruthLens 🕵️‍♂️

A mobile app that brings attention to underreported global news and injustice.

## 📱 Features
- Splash screen with globe animation
- Navigation between screens
- News feed coming soon...

- Clone this repo
- Run `npm install`
- Start with `npx expo start`
